## 📖 What is Bandit?

[OverTheWire: Bandit](https://overthewire.org/wargames/bandit/) is a wargame designed for **absolute beginners** in Linux and security.
Each level requires you to find a password hidden on a remote server using Linux commands, file manipulation, and creative problem solving.

It's not just about the commands — it's about **learning how to think like a hacker**.

## 💡 My Approach

I focused on **understanding over copying**. For every level I ask:
- *What is this level actually teaching?*
- *What would happen if I didn't know this command in a real pentest?*
- *What's the real-world application of this concept?*
- *What is the hint of this level?*

That's why some writeups are longer than the "solution" — the journey matters.
